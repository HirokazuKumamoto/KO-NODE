# KO-NODE
Knockout and Node.js(express) template of Grunt

## Preparation
### Install Node.js

https://nodejs.org/

### Install Node Package

npm install -g grunt-cli

## Create New Project

if you wanto to create new Project, path(C:\project\test)

mkdir C:\project\test

cd C:\project\test

grunt-init KO-NODE

npm install

grunt resolve
